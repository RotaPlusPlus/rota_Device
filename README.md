## Hardware Environment
#### Arduino
We tested on Genuino 101（Arduino 101 in US）

#### Sensor
Operation in DHTxx sensors (xx = 11-21-22-33-44) as a Humidity Sensor, we have seen.
http://playground.arduino.cc//Main/DHTLib

Arduino IDE on OSX
